# universal-app
boilerplate de aplicación que usa React y renderiza en el navegador y servidor

#Stack
- Express
- React
- Mobx
- Webpack
- Babel

#Como usar
1. Clona el repositorio
2. npm install
3. npm run start (para correr servidor express)
4. npm run webpack (para correr el servidor webpack - solo para entorno de desarrollo)

(Nota) no usar para producción
